Вы — начинающий разработчик в небольшой команде, которая делает онлайн-витрину электроники.

Дизайнеры уже подготовили макеты, менеджеры выбрали товары, а ваша задача — создать простой каталог товаров, чтобы фронтенд-разработчики могли получать данные и отображать их на сайте.

На этом этапе нужен минимальный функционал:
  - создать товар,

  - получить список всех товаров,

  - получить товар по его ID.

----

# Задача

Сущность только одна:

**Product**
  - id (UUID)
  - name (string)
  - price (number)

## Реализовать 3 эндпоинта

### 1. Создать товар

```bash
POST /products
```
Request:
```json
{
  "name": "Laptop",
  "price": 1500
}
```
Если нет name или price, вернуть ошибку 400, а иначе:  
Response (201 Created):
``` json
{
  "id": "UUID",
  "name": "Laptop",
  "price": 1500
}
```

### 2. Получить список товаров
```bash
GET /products
```
Response (200):  
```json
{
  "products": [
    {
      "id": "UUID",
      "name": "Laptop",
      "price": 1500
    }
  ]
}
```

### 3. Получить товар по ID
```bash
GET /products/{id}
```
Если товара нет, вернуть ошибку 404, а иначе:  
Response (200):  
```json
{
  "id": "UUID",
  "name": "Laptop",
  "price": 1500
}
```

# Формат ошибок (обязательный!)
ЛЮБАЯ ошибка должна возвращаться в формате:  
```json
{
  "error": "Product not found"
}
```

```bash
make run
```
